To install Scrapy on Ubuntu (or Ubuntu-based) systems, you need to install these dependencies:
sudo apt-get install python3 python3-dev python3-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev

Create a virtual environment
 $pip install virtualenv
 $virtualenv my_env

Activate a the virtual env
 $cd my_env
 $source bin/activate

Install Scrapy
 $pip install scrapy

To start a scrapy new project this will create a number of files
 $start scrapyproject stack
 $cd stack

From those files, now updated the items.py & create scrapy_spider_example.py file to extract the content in the spider directory
scrapy_spider.py contains the entire general example

To crawl
 $scrapy crawl example
